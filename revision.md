Write a Python function called process_numbers(nums) that takes a list of integers and returns a dictionary with the following keys:

"even" → a list of all even numbers

"odd" → a list of all odd numbers

"squares" → a list containing the square of each number

"greater_than_10" → a list of numbers greater than 10

"lambda_sum" → the sum of all numbers using a lambda + reduce

"iterator_first_three" → the first three items using an iterator (not slicing)

Additional rules:

You must use:

a for loop

an if/elif structure

a lambda function

an iterator created with iter()

Do not use list comprehensions.
